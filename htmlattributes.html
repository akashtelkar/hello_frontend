HTML Attributes

href : This attribute refer to a URL of any page in reference

src : This attribute points to a path

width : This attribute defines the width of the image

hight : This attribute defines the hight of the image

alt : This attribute defines alternate text to an image

style : This attribute is used to style an element, used for CSS

background-colour : This property adds colour to the background

color : This property adds color to the text

font-family : This property will define the font for the text

font-size : This property increases the size beyond H1 tag

text-align : This property will align the text

border : This property adds border to the element

padding : This property will pad the element

margin : This property defines margine

target : This property is use with a tag to where will the href URL will open

lang : This attribute defines language in HTML tag

title : This attribue adds extra information about the element
